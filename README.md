# 🧠 老板脑回路模拟器 · Boss Brain Simulator

一个 Claude Code Skill + 百炼 CLI (`bl pipeline`) 双形态工具，帮职场人做向上管理。

**核心能力：** 模拟老板看到一件事后的真实反应，找出表达漏洞，给出可以直接用的话术。复杂决策自动启动「五人决策小组」（巴菲特 / 马斯克 / 德鲁克 / 芒格 / 格鲁夫）。

---

## 两种用法

### 用法一：Claude Code Skill（推荐日常使用）

把 `SKILL.md` 放到你的 Claude Code skills 目录：

```bash
cp SKILL.md ~/.claude/skills/boss-brain-simulator/SKILL.md
```

重启 Claude Code 后，描述任何向上沟通场景即可自动触发。支持交互式追问、快速/标准/深度模式切换、高压问答演练。

### 用法二：百炼 CLI Pipeline（批量/演示用）

```bash
# 直接传问题
./run.sh "项目延期了，怎么跟老板解释"

# 用 problem.json
./run.sh

# 对话模式
./run.sh --chat
```

> Pipeline 版会一次性输出完整分析（标准模式 + 五人决策小组），不支持中途追问。

## 老板脑回路模拟器输出（标准模式）

1. **老板听完后的第一反应**
2. **老板真正关心的三件事**
3. **老板可能会追问什么**
4. **你现在的说法有什么问题**
5. **建议你怎么处理这件事**
6. **你需要老板做什么**
7. **你应该怎么说**（一句话版 / 30 秒口头版 / 微信钉钉版 / 正式汇报版）
8. **老板追问演练**（带 ✅⚠️❌ 安全标注）

---

## 五人决策小组输出（深度模式）

| 决策者 | 视角 |
|--------|------|
| 💰 巴菲特 | 资本与风险 |
| ⚡ 马斯克 | 第一性原理与效率 |
| 🏢 德鲁克 | 组织与顾客 |
| 🔍 芒格 | 反向思考与决策偏差 |
| 🎯 格鲁夫 | 执行、瓶颈与结果管理 |

追加输出：
- 五人各自判断
- 分歧与共识
- **推荐结论**（不能停留在「各有利弊」）
- **未来 14 天行动清单**（表格）
- **🚨 止损信号**
- **📊 验证指标**
- 规则触发检查

---

## 五人决策规则

1. 五个人不能重复表达相同观点
2. 每个人必须从自己的决策领域提出判断
3. 不得为了制造冲突而虚构问题，但必须主动寻找合理的反对意见
4. 当五人意见一致时，芒格必须提出最强反方观点
5. 当方案过于复杂时，马斯克必须提出删减方案
6. 当价值或付费方不清晰时，德鲁克和巴菲特可以建议暂缓
7. 当结论正确但无法执行时，格鲁夫必须指出责任人、时间或验收机制缺失
8. 最终不能停留在「各有利弊」，必须形成推荐结论

## 文件说明

| 文件 | 用途 |
|------|------|
| `SKILL.md` | Claude Code Skill 完整定义 |
| `system-prompt.md` | 单 prompt 对话版（用于 `--chat`） |
| `pipeline.yaml` | 2 步链式流水线（老板分析 → 五人决策） |
| `workflow.yaml` | 14 步并行流水线（8 节老板分析 + 5 人独立判断 + 收敛） |
| `problem.json` | 问题输入模板 |
| `run.sh` | 启动脚本 |
| `README.md` | 本文件 |
